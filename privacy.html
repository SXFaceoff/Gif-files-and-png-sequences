1. Кто собирает данные

Данные обрабатываются в рамках работы бота «Лилит» на канале SX_Faceoff. Владелец: SX_Faceoff. Бот работает локально и не является публичным сервисом.

2. Какие данные собираются и на каком основании

Данные Основание Как отказаться
Никнейм (имя пользователя на платформе Trovo), дата последнего визита Законный интерес (модерация) Данные модерации могут храниться в течение разумного срока. Удаление возможно по запросу, если это не нарушает безопасность канала.
История нарушений правил канала Законный интерес (модерация) Данные модерации могут храниться в течение разумного срока. Удаление возможно по запросу, если это не нарушает безопасность канала.
Баланс SX-коинов, статистика игр Явное согласие (!игрок) !отмена игрок — полное удаление
Контекст разговоров с ИИ, якоря памяти Явное согласие (!разрешаю ИИ) !отмена ИИ — полное удаление

3. Принцип явного согласия

Ни одна функция, связанная с хранением игровых или персональных данных, не активируется без явного действия со стороны пользователя. При вводе команды !игрок бот запрашивает подтверждение и объясняет, какие данные будут сохраняться. Только после ответа !да данные начинают собираться.

Первичное уведомление о сборе данных и доступных командах управления отображается при первом взаимодействии пользователя с ботом.

4. Как используются данные

  • Данные используются исключительно для работы функций бота на канале SX_Faceoff
  • Данные не передаются и не продаются третьим лицам
  • Сообщения чата передаются в Google Gemini API исключительно при прямом взаимодействии пользователя с ИИ (обращение к боту или команды ИИ-функций) — в соответствии с условиями Google

5. Хранение данных

Все данные хранятся локально на устройстве владельца канала в базе данных SQLite. Данные не размещаются на сторонних серверах и не передаются в облако.

6. Удаление данных

  • !отмена игрок — удаляет статус игрока, баланс коинов и всю игровую статистику
  • !отмена ИИ — удаляет историю диалогов, якоря памяти, предпочтения (фаза 2)
  • Данные модерации (история нарушений) хранятся в течение разумного срока для обеспечения безопасности канала и могут быть удалены по запросу, если это не нарушает безопасность канала

Для запроса на удаление данных обратитесь напрямую через чат канала.

7. Уведомление о сборе данных

Первичное уведомление о сборе данных отображается при первом взаимодействии пользователя с ботом. Бот периодически напоминает в чате о доступных командах управления данными. Правила канала, отображаемые при входе, содержат информацию о системе согласия.
