Политика конфиденциальности / Privacy Policy

Последнее обновление: апрель 2025 г.

1. Какие данные собираются

Бот «Лилит» может сохранять следующие данные зрителей канала:

  • Имя пользователя Trovo — для идентификации в играх и системе баланса
  • Игровой баланс (SX-coins) — внутренняя валюта, используемая только на канале
  • Сообщения в чате — временно обрабатываются для ответов ИИ (не сохраняются постоянно)

2. Как используются данные

  • Данные используются исключительно для работы функций бота на канале SX_Faceoff
  • Данные не передаются третьим лицам и не используются в коммерческих целях
  • Сообщения чата передаются в Google Gemini API для генерации ответов ИИ — в соответствии с условиями Google

3. Хранение данных

Игровые данные (баланс, статистика) хранятся локально на устройстве владельца канала в базе данных SQLite. Данные не хранятся на сторонних серверах.

4. Удаление данных

Вы можете запросить удаление ваших данных, написав в чат канала или связавшись с владельцем через платформу Trovo.
